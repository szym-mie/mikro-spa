Why JS

Hello world!

JavaScript is a primary way to make your sites behave dynamic.

Since its introduction in 1995, the language has been evolving constantly to better suit the needs of web developers all over the world. This makes JavaScript a very flexible tool, further augmented by endless amount of available frameworks such as React, Angular and Vue; and many built-in features known as Web API.

Not only browser

For a long time JavaScript has been only available in browsers. You had to use different languages for everything backend related. This is no more. With the power of Node.js you can also design and deploy your own servers. While at first it might seem like a completely different world, once you get a hold of JavaScript learning backend frameworks such as ExpressJS will be a breeze.

Native apps

But what if you could make an app in JavaScript and have it behave like native app? Well, you can now with React Native and ElectronJS. Both provide great ways to develop native apps both for desktop and mobile.
