...or convert to an array using spread operator! You can also use spread to create variadic argument functions. 

const numberArray = Array.from(new Array(10), (_, index) => index)
const doubleArray = numberArray.map(n => n * 2)
const evenArray = numberArray.filter(n => n % 2 == 0)
const sumOfArray = numberArray.reduce((acc, n) => acc + n)
const remainderObject = Object.groupBy(numberArray, n => n % 3 + 1)

Javascript offers a lot in terms of functional programming. You can easily write concise code with just arrays and objects. 

const url = 'https://wdai-mikro-szym-mie.netlify.app/endpoint/users.json'
const headers = { 'Content-Type': 'application/json' }
fetch(url, { headers })
.then(response => response.json())
.then(users => { alert(
    'revid' + users.revid + '\n' + 
    'users' + users.map.reduce((acc, {uid, firstname, lastname}) => 
        uid + ' ' + firstname + ' ' + lastname + '\n'
    ) 
) })
.catch(err => { alert(err) })
